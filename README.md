# Family Gallery

This is a simple **HTML & CSS family gallery** with circular thumbnails.  
Clicking on a thumbnail opens a **full-size image** in a popup.

## Features
- Circular thumbnail images
- Popup view of full-size images
- Pure HTML & CSS (no JavaScript)
- Responsive design

## How to view
1. Open the `index.html` file in your browser.  
2. Or visit the live site: [https://tslalomHa.github.io/our-family-gallery/](https://tslalomHa.github.io/family-gallery/)
